API ENDPOINTS

將以下 endpoint 包成 ADK FunctionTool 給您的 Agent

GET /api/products 列出所有產品

Query 參數:?category=柱材 可過濾

curl http://localhost:8080/api/products?category=樑材
GET /api/products/{code} 取得單一產品 
curl http://localhost:8080/api/products/HW300x300
GET /api/grades 列出材質與單價 
curl http://localhost:8080/api/grades
POST /api/quotes 計算報價(支援議價:係數 / 折讓 / 加成) 
curl -X POST http://localhost:8080/api/quotes \
  -H "Content-Type: application/json" \
  -d '{
    "items":[{"product_code":"HW300x300","grade":"SS400","length_m":6,"quantity":20}],
    "manual_discount_factor": 0.95,
    "manual_concession_ntd": 5000
  }'

手動議價:0 < factor ≤ 1;concession ≥ 0;surcharge ≥ 0;後兩者互斥。

POST /api/quotes/match 反向:給目標一口價,系統反推折讓 or 加成 
curl -X POST http://localhost:8080/api/quotes/match \
  -H "Content-Type: application/json" \
  -d '{
    "items":[{"product_code":"HW300x300","grade":"SS400","length_m":6,"quantity":20}],
    "target_final_ntd": 280000
  }'
POST /api/quotes/save 確認後存進 DB,回傳 quote_id(用於 Memory Bank) 
curl -X POST http://localhost:8080/api/quotes/save \
  -d '{"customer":"明陽營造","project":"板橋廠案","items":[...],"note":"客戶一口價需求"}'
GET /api/quotes 列已存報價(可 ?customer= 或 ?sales_user_id=)
GET /api/quotes/{id} 取單一已存報價(完整 items + 議價軌跡)
